Introducción a HTML

HTML es el lenguaje de marcado utilizado para crear la estructura de una página web.

¿Qué significa HTML?

HTML significa HyperText Markup Language, que en español se conoce como Lenguaje de Marcado de Hipertexto.

¿Para qué sirve?

HTML permite organizar el contenido de una página por medio de títulos, párrafos, enlaces, imágenes, tablas y formularios.

Estructura básica

Un documento HTML contiene la declaración del tipo de documento, la etiqueta principal HTML, una sección de encabezado y el cuerpo donde se coloca el contenido visible.

Lo que aprendí

En esta práctica aprendí qué es HTML, para qué se utiliza y cuáles son las partes principales de un documento HTML.
