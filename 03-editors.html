Editores HTML

Un editor HTML es un programa que permite crear y modificar archivos con extensión .html.

Editor utilizado

Para desarrollar estas prácticas estoy utilizando Visual Studio Code, ya que facilita la escritura del código mediante resaltado de sintaxis y sugerencias automáticas.

Proceso para crear una página web

  1. Abrir Visual Studio Code.
  2. Crear un archivo con extensión .html.
  3. Escribir el código HTML.
  4. Guardar el archivo.
  5. Abrirlo en un navegador para visualizar el resultado.

Lo que aprendí

En esta lección comprendí la importancia de utilizar un editor de código para crear páginas web de forma organizada y eficiente.
