Elementos de HTML

En HTML todo el contenido está formado por elementos. Cada elemento cumple una función diferente dentro de la página.

Ejemplos de elementos

Elemento de encabezado

Los encabezados ayudan a organizar la información y hacen que la página sea más fácil de leer.

Elemento de párrafo

Los párrafos permiten escribir información o descripciones.
